# SlugSwap Landing Pages v2.0 — BREATHTAKING Designs

3 UNFORGETTABLE, production-ready React landing pages that make people say "WOW".

## 🔥 The Designs

### 1. **Hero Bold** (`/hero-bold`)
**Brutalist Editorial**
- LOUD, confident, unapologetic brutalism
- Impact font, thick black borders, sharp shadows
- Yellow/Blue/Red color palette
- Stacked giant text, geometric chaos
- **Vibe**: In your face. Impossible to ignore.

### 2. **Maximalist Chaos** (`/maximalist`)
**Overwhelming Visual Richness**
- Layered elements, 50+ animated shapes
- Clashing colors: cyan, yellow, red, purple, green
- Impact + Arial Black fonts
- Overlapping cards, diagonal rotations
- **Vibe**: Early web meets modern design on steroids. Controlled chaos.

### 3. **Collision Grid** (`/collision`)
**Asymmetric Grid Disruption**
- Radical 60/40 diagonal split with overlapping zones
- Burn orange + electric teal color conflict
- Archivo Black + Work Sans typography pairing
- Elements break boundaries with aggressive layout energy
- **Vibe**: Tension, impact, and controlled visual collision.

## 🎨 Design Philosophy

Each landing page follows these principles:

1. **Commit to the aesthetic** — Pick a BOLD direction and execute it fully
2. **Unique typography** — No Inter, Roboto, or system fonts
3. **Unexpected layouts** — Asymmetry, overlap, diagonal flow
4. **Atmospheric backgrounds** — Textures, gradients, animated elements
5. **High-impact animations** — Staggered reveals, morphing shapes, glitch effects
